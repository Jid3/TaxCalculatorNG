# TaxCalculatorNG

A Calculator for the new tax laws implemented by the Nigerian government on 1st Jan, 2026.

## Features
- **Optimized Performance:** Image compression and parallel uploads for profile pictures.
- **Fast Loading:** Stale-while-revalidate caching for tax history.
- **Smart Formatting:** Automated comma formatting for numerical inputs.
- **Instant Response:** Optimistic UI updates for profile and tax saves.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```
2. Locate folder

   ```bash
   cd Taxcalculator
   ```

4. Start the app

   ```bash
   npx expo start
   ```
